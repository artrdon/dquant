## 2. Installation

### 2.1 Requirements

- Python 3.7 or higher
- pip (Python package manager)

**Dependencies (installed automatically):**
- `cycler >= 0.11.0`
- `joblib >= 1.2.0`
- `lightgbm >= 3.3.0`
- `matplotlib >= 3.5.0`
- `numpy >= 1.21.0`
- `onnx >= 1.14.0`
- `onnxruntime >= 1.15.0`
- `pandas >= 1.5.0`
- `scikit-learn >= 1.2.0`
- `skl2onnx >= 1.14.0`
- `xgboost >= 1.7.0`
- `onnxconverter-common >= 1.9.0`
- `onnxmltools >= 1.11.0`

### 2.2 Installation via pip

```bash
pip install dquant
```

### 2.3 Verifying Installation

```python
import dquant
print(dquant.__version__)
# Should output the version
```
